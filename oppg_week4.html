Du får oppgitt 3 funksjoner:

\( f = (x+y)ln(\frac{x}{y}) \)

$ g = ln(\frac{x}{y}) + \frac{x + y}{x} $

$ h = \frac{1}{y} + \frac{1}{y} $

a) Hvilke(n) av disse funksjonene kan sies å være "Euler-homogene" funksjoner? Gi en matematisk begrunnelse.

Let f(x,y) be a homogeneous function of order n so that :

$ f(tx,ty)=t^nf(x,y). $

Prøver så å sette inn i likningene:

$ f(tx,ty) = (tx + ty) ln(\frac{tx}{ty}) = t f(x,y) $

Så denne stemmer for n = 1

$ g(tx,ty) = ln(\frac{tx}{ty}) + \frac{tx + ty}{tx} = t^0 g(x,y) $

Så denne stemmer for n = 0

$ h(tx,ty) = \frac{1}{ty} + \frac{1}{tx} = \frac{1}{t} f(x,y) = t^{-1} f(x,y) $

Så denne stemmer for n = -1
